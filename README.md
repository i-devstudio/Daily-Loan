# DebtFlow (ระบบบริหารลูกหนี้รายวัน - WebApp)

ยินดีต้อนรับสู่ **DebtFlow** ระบบบริหารจัดการลูกหนี้รายวันสำหรับเจ้าหนี้รายย่อย ออกแบบมาเป็นพิเศษในรูปแบบ Mobile Web App (Responsive) ที่ทำงานร่วมกับฐานข้อมูล Google Sheets และบริการเก็บรูปภาพ Google Drive ผ่านระบบหลังบ้าน Google Apps Script (GAS) 

แอปพลิเคชันนี้ได้รับการจัดประเภทโครงสร้างใหม่ให้พร้อมพัฒนาและรันเป็น **Vite Web Application** เพื่อการรันเซิร์ฟเวอร์แบบ Local ที่ราบรื่น รวดเร็ว และเป็นสากล

---

## 🌟 ฟีเจอร์หลัก (Features)
1. **Dashboard สรุปภาพรวม:** แสดงจำนวนลูกหนี้ทั้งหมด, ยอดเงินต้นคงเหลือทั้งหมด, ยอดเก็บเงินวันนี้ และยอดเก็บเงินประจำเดือนนี้
2. **ระบบลูกหนี้ด่วน:** ค้นหารายชื่อหรือเบอร์โทรลูกหนี้ได้แบบ Real-time บนการ์ดแสดงสถานะที่ออกแบบมาเป็นมิตรกับสมาร์ทโฟน
3. **คำนวณเงินปันผล/ดอกเบี้ยรายวัน:** คำนวณดอกเบี้ยสะสมขั้นต่ำอัตโนมัติตามหลักการทางธุรกิจ
4. **ตรรกะการรับชำระเงินที่ถูกต้อง (Business Logic):** เมื่อชำระเงิน ระบบจะหักออกเป็น **"ค่าดอกเบี้ยรายวันก่อน"** เสมอ และส่วนที่เหลือจึงนำไป **"หักเงินต้น"** พร้อมแสดงตัวอย่างผลลัพธ์ (Preview) ให้เห็นทันทีก่อนยืนยันการชำระเงิน
5. **ระบบป้องกันการชำระล้นหนี้ (Overpayment Capping):** หากผู้กู้มียอดชำระเงินเกินภาระหนี้ที่มีอยู่ทั้งหมด (เช่น ยอดต้นค้าง 5,000 + ดอกวันนั้น 200 = หนี้รวม 5,200 แต่ลูกหนี้กรอกจ่าย 6,000) ระบบจะทำการคำนวณและหักชำระจริงเพียง **5,200 บาท** (หักดอกเบี้ย 200, หักต้น 5,000 และคงเหลือใหม่เป็น 0) พร้อมส่งคำเตือนในป๊อปอัปให้ผู้ใช้รับรู้ทันทีโดยอัตโนมัติ
6. **ระบบเอกสารแนบในตัว:** รองรับการอัปโหลดและแสดงผลรูปถ่ายบัตรประชาชน ทะเบียนบ้าน รูปหน้าบ้าน และรูปลูกหนี้ ขนาดสูงสุด 5MB/รูป (จัดเก็บอย่างเป็นระเบียบลง Google Drive ในบัญชีของท่านเอง)
7. **ระบบความปลอดภัยล็อกอิน:** เข้าระบบด้วยรหัส PIN 4 หลัก ปรับปรุงเปลี่ยนรหัสได้ง่าย
8. **ทำงานแบบออฟไลน์/ทดลองใช้ก่อนผูกฐานข้อมูล:** หากไม่มีการตั้งค่า API ระบบจะเข้าสู่ **"โหมดจำลอง (Mock Data)"** อัตโนมัติ ให้ทดสอบใช้ระบบได้ทันทีโดยไม่ต้องเชื่อมต่อใดๆ

## 🚀 ขั้นตอนการติดตั้งและรัน WebApp

### วิธีที่ 1: การเปิดใช้งานผ่าน Vite (แนะนำสำหรับการพัฒนาและใช้งานบน Desktop)
1. เปิดโปรแกรม Terminal หรือ Command Prompt ไปที่โฟลเดอร์นี้
2. รันคำสั่งต่อไปนี้เพื่อติดตั้ง Vite:
   ```bash
   npm install
   ```
3. รันระบบเซิร์ฟเวอร์จำลองด้วยคำสั่ง:
   ```bash
   npm run dev
   ```
4. ระบบจะทำการเปิดหน้าแอปบนเบราว์เซอร์ให้อัตโนมัติที่ `http://localhost:3000` (หรือสามารถพิมพ์ URL นี้ลงไปเองได้)

### วิธีที่ 2: การดับเบิลคลิกเปิดไฟล์ธรรมดา
ท่านสามารถดับเบิลคลิกเปิดไฟล์ **[index.html](file:///Users/raomark/Desktop/Daily%20Loan%20Management%20System/index.html)** บนคอมพิวเตอร์เพื่อทดสอบระบบได้ทันทีโดยไม่ต้องรันผ่าน Node.js

## 🔗 ขั้นตอนผูกฐานข้อมูล Google Sheets

1. สร้าง Google Sheet ใหม่ขึ้นมา 1 ไฟล์
2. คลิกเมนูด้านบนเลือก **ส่วนขยาย** (Extensions) -> **แอปสคริปต์** (Apps Script)
3. นำโค้ดในไฟล์ [backend.gs](file:///Users/raomark/Desktop/Daily%20Loan%20Management%20System/backend.gs) ไปคัดลอกวางทดแทนทั้งหมดและกดบันทึก
4. กดปุ่ม **การทำให้ใช้งานได้** (Deploy) -> **การทำให้ใช้งานได้ใหม่** (New deployment)
   - เลือกประเภทเป็น **เว็บแอป** (Web App)
   - เรียกใช้ในฐานะ **ตัวคุณเอง**
   - ผู้มีสิทธิ์เข้าถึงเลือก **ทุกคน** (Anyone)
5. ทำการคัดลอก **URL ของเว็บแอป** ที่ได้ ไปป้อนใส่ในหน้า **ตั้งค่า** (Settings) ของแอปที่รันอยู่ในเบราว์เซอร์ แล้วกด **เชื่อมต่อ API**
6. ป้ายปุ่มเชื่อมต่อด้านบนจะแสดงไฟสีเขียว **API Connected** ข้อมูลทั้งหมดจะวิ่งตรงสู่ Google Sheet บนบัญชีคลาวด์ของคุณทันที!
